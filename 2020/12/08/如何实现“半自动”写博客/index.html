如何实现“半自动”写博客

Posted on 2020-12-08 Views:

像某些懒人，就是懒得打那几行代码（比如我）
所以花了亿点点时间，浅浅学习了一下.bat来完成这个过程

Create.bat

@echo off
set /p name=input name:
echo name:%name%
echo please wait
hexo new post %name% && call openIDE.bat %name%
pause

分析：

  1. @echo off执行以后，后面所有的命令均不显示，
    包括本条命令。（echo 表示显示此命令后的字符）
  2. 输入文档名
  3. 显示自己的文件名
  4. 显示“please wait”
  5. 使用Hexo指令创建新文档并唤起openIDE.bat（期间传递了变量%name%）
  6. 按任意键继续…

openIDE.bat

start /min /w mshta vbscript:setTimeout("window.close()",1000)
echo the input is %1
set name=%1
echo %name%
start /d "C:\Users\Nameless\NamelessUGhome\NamelessUG\source\_posts"    %name%.md
pause

分析：

  1. 等待1秒
  2. 这一行是完成完成变量传递可以用下图理解

  1. 输出名字（个人觉得没有用这一行回头试试删除）
  2. 打开对应文件
  3. 按任意键继续…

Hexo s -g.bat

hexo s -d

这条代码就是生成并启动本地服务，进行文章预览调试

Deploy.bat

start clean.bat
start /min /w mshta vbscript:setTimeout("window.close()",2000)
start gd.bat
exit
  1. 唤起clean.bat
  2. 等待2秒
  3. 唤起gd.bat
  4. 退出程序

Clean.bat

@echo off
echo clean
echo please wait
hexo clean
exit
  1. @echo off执行以后，后面所有的命令均不显示，
    包括本条命令。（echo 表示显示此命令后的字符）
  2. 显示clean
  3. 显示please wait
  4. 执行hexo clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)。
在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。

  1. 退出程序

gb.bat

start /min /w mshta vbscript:setTimeout("window.close()",2000)
hexo g -d
exit
  1. 等待2秒
  2. 文件生成后立即部署网站
  3. 退出程序

总结

其实实现这个小小功能还是蛮简单（指借用别人代码），重点还是在理解每一行代码的作用（像我这样的渣渣）。
顺便安利一下坚果云的Markdown真不错很爽（对于新手）本来原文（指抄袭对象）是使用Sublime Text 3，然而小白可以偷懒何乐不为捏😀。
